Hệ Thống Lý Thuyết

1. Câu Điều Kiện & Unless

If ... not ⟷ Unless ... (Trừ khi)

2. Hiện Tại Hoàn Thành & Quá Khứ Đơn

The last time... was... ⟷ S + hasn't/haven't + V3... since...

3. So sánh (Equal/Comparative/Superlative)

S1 + V + Adj-er + than + S2 ⟷ S2 + V(not) + as + Adj + as + S1

4. Cấu trúc Suggestion (Gợi ý)

Why don't we/you...? ⟷ S + suggested + (that) + S + should + V(bare)

5. Cấu trúc Cảm xúc / Sở thích

Like / Enjoy / Interested in ⟷ Be keen on / fond of / Take an interest in

6. Cấu trúc "It takes"

S + spend + [time] + V-ing ⟷ It takes + [someone] + [time] + to V-inf

7. Cấu trúc Cause & Effect (Because/Although)

Because/Although + Clause ⟷ Because of/In spite of + Noun Phrase/V-ing

8. Giới từ sau Động từ (Nominate...)

Nominate someone as + [vị trí] / Nominate someone for + [giải thưởng]

9. Cấu trúc bắt buộc (Must/Compulsory)

S + must + V ⟷ It is compulsory for someone to V

10. Cấu trúc So... that / Such... that

So + Adj + that ⟷ Such + (a/an) + Adj + N + that

11. Đại từ quan hệ (Who/Which/That)

Dùng để nối hai câu đơn có cùng đối tượng hoặc thay thế danh từ đứng trước.

12. Câu ước (Wish)

Ước trái ngược hiện tại: S + wish(es) + S + V2/ed (were)

13. Câu tường thuật (Reported Speech)

Lùi thì và đổi trạng từ: "I am ... now" → She said she was ... then

14. Cấu trúc Difficult / Find it

Find it easy/difficult to V ⟷ Have no difficulty (in) V-ing

15. Phrasal Verbs (Động từ ghép)

Ví dụ: Go wrong ⟷ Break down (hỏng hóc)
